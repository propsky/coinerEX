# 04-開發文件

## 資料夾說明
本資料夾包含具體的開發工作清單和實作指引。

## 文件清單

### 雲服務開發工作清單.md
- **用途**: 後端雲服務開發任務規劃
- **對象**: 後端開發工程師、專案經理
- **內容**: 
  - 18項開發任務 (12週，228小時)
  - 基於現有UI不重做的策略
  - Linode伺服器部署規劃
  - API開發與MQTT整合

### IPC程式開發工作清單.md
- **用途**: 硬體端IPC程式開發規劃
- **對象**: 嵌入式系統工程師
- **內容**:
  - 13項開發任務 (6週，164小時)
  - Nano Pi M1plus + Python開發
  - 硬體通訊與感應器整合
  - MQTT客戶端實作

## 開發重點
- 雲服務：專注後端API，沿用現有前端UI
- IPC程式：硬體控制與雲端通訊整合
- 兩個清單可並行開發
- 包含詳細的時程規劃與技術規格

## 使用說明
- 開發團隊按照清單逐項完成
- 每個任務都有預估工時參考
